Bubble Pop Chain Menu Tile Icons
License: AI-generated via Meshy AI (Text to Image), post-processed locally (chroma-key transparency, trim, resize). Owned/licensed for this project's use.
Runtime: Local PNG files, no remote loading.
Style: Glossy white/gray 3D-rendered glyphs (colour-neutral) so they read cleanly against any of the main-menu tile's own colour-gradient badge background. Used by the main-menu `.tile-ic` badges (index.html) in place of the previous inline currentColor SVG line icons.
chapter-*.png (chapter-meadow/frosty/thunder/crystal/cosmic for the 5 authored campaign chapters, chapter-aurora/ember/tidal/verdant/obsidian/solar/nebula/mirage for the 8 procedural late-game chapters) are full-colour "world" badges shown on the level map's chapter headers, rendered via `ui.js` `chapterIconHtml(chapter)` in place of the plain emoji those headers used to show (see levels.js CHAPTERS/PROC_CHAPTERS). chapter-ember.png is a plain flame emblem rather than a literal glowing coal/ember rock — an ember-rock render came out pale/washed-out twice in a row (a real model limitation), so a flame (which rendered vividly) was used instead.
