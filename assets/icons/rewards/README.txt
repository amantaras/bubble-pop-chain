Bubble Pop Chain Reward Icons
License: AI-generated via Meshy AI (Text to Image), post-processed locally (chroma-key transparency, trim, resize). Owned/licensed for this project's use.
Runtime: Local PNG files, no remote loading.
Style: Glossy full-colour 3D-rendered icons for reward types shared across many screens (calendar, wheel, quests, achievements chest reveal, win-screen bonus choices, level map, falling events, level briefing, shop, puzzles). crate.png is the pet-crate reward icon, rendered via `ui.js` `crateIconHtml`. dust.png is the Pet Dust reward icon, rendered via `dustIconHtml`. season-xp.png is the Season XP reward icon, rendered via `seasonXpIconHtml`. gift.png is the shared "gift"/treasure icon (falling gift event token, treasure milestone badge, chest-reveal modal, collect-all sweep), rendered via `giftIconHtml`. warning.png is the falling "problem" event token icon, rendered via `warningIconHtml`. boss.png is the generic boss milestone badge icon on the level map, rendered via `bossIconHtml`. boss-frozen.png/boss-stone.png/boss-color.png/boss-vine.png are per-archetype boss portraits shown on the level briefing screen for boss milestone levels, rendered via `bossPortraitHtml(kind)` (see milestones.js `bossConfig` for the archetype `kind`). starter-pack.png/piggy-bank.png/no-ads-shield.png are the Shop screen's Starter Pack bundle, Piggy Bank card, and Remove Ads offer icons, rendered via `starterPackIconHtml`/`piggyBankIconHtml`/`noAdsIconHtml` (replacing the plain 🎁/🐷/🚫 emoji those rows used to show). puzzle-clear.png/puzzle-frozen.png/puzzle-stone.png/puzzle-color.png are the Puzzles screen's per-objective-type badges, rendered via `puzzleTypeIconHtml(type)` (see puzzle.js `puzzleTypeMeta`). trophy.png/palette.png/paw.png/hammer.png are the win-screen reward-card icons for the "best score"/"theme unlock"/"pet"/"tool" reward kinds, rendered via `trophyIconHtml`/`paletteIconHtml`/`pawIconHtml`/`hammerIconHtml` and wired into `_rewardCardIcon`'s per-kind map (replacing the plain 🏆/🎨/🐾/⚒️ emoji; the "boss" kind reuses the existing `bossIconHtml`). flame.png is the daily-streak icon, rendered via `flameIconHtml` (menu daily summary "N🔥" and the win-screen "Streak N" reward card). snowflake.png is the streak-freeze-token icon, rendered via `snowflakeIconHtml` (menu daily summary freeze count and the win-screen "Freeze earned/used" reward card, replacing the plain ❄️ emoji). dice.png is the Daily Wager result icon, rendered via `diceIconHtml` (win-screen "Wager won/lost" reward card, replacing the plain 🎲 emoji). egg.png/chick.png are the Pet Store's Mystery Egg incubation icons, rendered via `eggIconHtml`/`chickIconHtml` (replacing the plain 🥚/🐣 emoji on the crate-egg-icon card). party.png is a celebration icon, rendered via `partyIconHtml` (the Pets screen's Party tab and the Lucky Wheel's jackpot segment, replacing the plain 🎉 emoji). All mirror the same local-image pattern as `coinIconHtml`.
