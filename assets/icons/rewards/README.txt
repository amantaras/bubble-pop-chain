Bubble Pop Chain Reward Icons
License: AI-generated via Meshy AI (Text to Image), post-processed locally (chroma-key transparency, trim, resize). Owned/licensed for this project's use.
Runtime: Local PNG files, no remote loading.
Style: Glossy full-colour 3D-rendered icons for reward types shared across many screens (calendar, wheel, quests, achievements chest reveal, win-screen bonus choices, level map, falling events, level briefing). crate.png is the pet-crate reward icon, rendered via `ui.js` `crateIconHtml`. dust.png is the Pet Dust reward icon, rendered via `dustIconHtml`. season-xp.png is the Season XP reward icon, rendered via `seasonXpIconHtml`. gift.png is the shared "gift"/treasure icon (falling gift event token, treasure milestone badge, chest-reveal modal, collect-all sweep), rendered via `giftIconHtml`. warning.png is the falling "problem" event token icon, rendered via `warningIconHtml`. boss.png is the generic boss milestone badge icon on the level map, rendered via `bossIconHtml`. boss-frozen.png/boss-stone.png/boss-color.png/boss-vine.png are per-archetype boss portraits shown on the level briefing screen for boss milestone levels, rendered via `bossPortraitHtml(kind)` (see milestones.js `bossConfig` for the archetype `kind`). All mirror the same local-image pattern as `coinIconHtml`.
