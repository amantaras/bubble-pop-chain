Bubble Pop Chain Pet Trait Badge Icons
License: AI-generated via Meshy AI (Text to Image), post-processed locally (chroma-key transparency, trim, resize). Owned/licensed for this project's use.
Runtime: Local PNG files, no remote loading.
Style: Glossy full-colour 3D-rendered icons for each pet personality trait (Balanced/Swift/Mighty/Lucky/Keen/Fiery — see pets.js TRAITS), shown in the pet detail pane's trait badge via `ui.js` `traitIconHtml`, replacing the plain 🔘/⚡/💪/🍀/🎯/🔥 emoji. Keen uses a golden arrow instead of a literal bullseye target — target/bullseye renders came out pale and washed-out across three separate Meshy attempts (a real model limitation for circular red/white targets), so a distinct precision-themed visual was used instead.
