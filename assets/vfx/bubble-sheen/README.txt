Bubble Sheen Overlay
License: AI-generated via Meshy AI (Text to Image), post-processed locally (chroma-key transparency, trim, resize). Owned/licensed for this project's use.
Runtime: Local PNG file, no remote loading.
Style: A colour-neutral grayscale glass-sphere lighting map (specular highlight, rim light, soft ambient shadow). Composited on top of every procedurally-recoloured bubble in renderer.js `drawBubbles` (blend mode "soft-light") to add a richer painterly highlight/shadow texture without needing per-theme/per-colour baked art — the underlying gradient fill still drives the actual per-theme colour.
