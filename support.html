Bubblit! Support

Help for the web, PWA, Android, and iOS versions.

If something goes wrong, include your platform, browser or device model, approximate level/mode, and what happened right before the issue.

Common Fixes

If the installed PWA shows old behavior, close it fully and reopen it, or refresh once. The service worker uses a network-first strategy, so reopening or refreshing should pull the newest build when you are online.

If audio is silent, tap once inside the game. Browsers and mobile platforms require a user gesture before game audio can start.

If progress appears missing, check whether you are using a different browser, profile, or device. Bubblit! saves progress locally and does not currently sync across devices.

Purchases And Ads

Store builds should only show paid or rewarded surfaces when the matching platform provider is available. If a purchase or rewarded ad is unavailable, try again later after updating the app. Never reinstall before contacting support if you need to preserve local progress.
